HTML/HTML5 CSS/CSS3 Javascript Responsive Design Parallax Scrolling

HTML/HTML5

HTML stands for HyperText Markup Language. It is used for making the structure of a website and also called the language of the web. It is also called the backbone of a webpage.

HTML uses tags to make various elements in a webpage. Developers can make the design of the website using HTML and can do the styling of the website by using CSS

CSS/CSS3

CSS stands for Cascading Style Sheet. While HTML helps in making the structure of the website, CSS helps in styling the website and make it look good.

CSS uses various properties to style the structure of a HTML element, ids or classes. CSS also helps in making the website responsive.

JavaScript

JavaScript is a only programming which is supported in the frontend of a website. It helps in form validation, styling a website and making the website interactive. It also helps in making frontend games.

JavaScript also supports backend programming with the help of its Node.js framework and can do all the works that any programming language can do using various frameworks.

Responsive Design

Responsive Design is the design that makes a website adaptable to any standard screen size. Responsive design helps in viewing a website on desktop, tablet and mobile screen.

Nowadays, it has become mandatory for a website to become response otherwise the mobile users are easily fed up with the annoying design of the website.

Parallax Scrolling

Parallax Scrolling is the feature that helps various elements of the webpage to scroll at different speed. When the user scrolls, two or more elements can move at various speed.

Parallax Scrolling make a website look good and attracts user to the website by its unique design and scrolling effects. It is not used in many website but it is getting popular very fast.
